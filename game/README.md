# Runner Game - Реализация

## Описание

Клон playable креатива на основе полного анализа референса.

## Структура проекта

```
game/
├── src/                    # Исходный код
│   ├── core/              # Ядро игры
│   ├── entities/          # Игровые сущности
│   ├── systems/           # Системы
│   ├── ui/                # UI компоненты
│   └── utils/             # Утилиты
├── assets/                 # Ассеты (для разработки)
├── build/                 # Скрипты сборки
└── index.html             # Финальный HTML файл
```

## Технологии

- **PixiJS** - рендеринг
- **GSAP** (CDN) - UI анимации
- **Howler.js** (встроен) - аудио
- **Vanilla JavaScript** - игровая логика

## Требования

- Единый HTML файл (< 5MB)
- Все ассеты встроены как data URI
- Responsive design
- Оптимизация для мобильных
