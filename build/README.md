# Скрипты сборки

## Описание

Скрипты для создания единого HTML-файла из модульной структуры проекта.

## Использование

### Базовая сборка
```bash
npm run build
```

Или напрямую:
```bash
node build/bundle.js
```

## Что делает скрипт

1. **Конвертирует ассеты** в base64 data URI
2. **Объединяет все JS файлы** из `game/src/`
3. **Извлекает CSS** из `game/index.html`
4. **Заменяет пути к ассетам** на data URI
5. **Создает единый HTML файл** `playable_game.html`

## Текущие ограничения

- ⚠️ Импорты ES модулей обрабатываются упрощенно
- ⚠️ Внешние библиотеки (PixiJS, Howler.js, GSAP) пока загружаются через CDN
- ⚠️ Код не минифицирован
