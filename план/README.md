## Быстрый старт

1. **Этап 1**: Настройка проекта и базовые системы
2. **Этап 2**: Реализация рендеринга и игровых сущностей
3. **Этап 3**: UI система и игровая логика
4. **Этап 4**: Полировка и оптимизация

---

## Основные принципы

- ✅ Единый HTML файл (< 5MB)
- ✅ Все ассеты встроены как data URI
- ✅ Минимум внешних зависимостей
- ✅ Оптимизация для мобильных устройств
- ✅ Responsive design

---

## Технологии

- **PixiJS** - рендеринг
- **GSAP** (CDN) - UI анимации
- **Howler.js** (встроен) - аудио
- **Vanilla JavaScript** - игровая логика

## Следующие шаги

### Рекомендуемый подход (с контролем каждого этапа):

1. **Начните с [09_incremental_steps.md](./09_incremental_steps.md)** - Поэтапный план
   - Каждый этап независим и тестируем
   - Видите прогресс на каждом шаге
   - Легко находите и исправляете баги

2. **Изучите структуру:**
   - [01_architecture.md](./01_architecture.md) - Архитектура
   - [01_structure.md](./01_structure.md) - Структура кода

3. **Следуйте поэтапному плану:**
   - Этап 0: Базовая инфраструктура ✅
   - Этап 1: Фон
   - Этап 2: Игрок (статичный)
   - Этап 3: Игрок (анимация бега)
   - Этап 4: Игрок (прыжок)
   - И так далее...
